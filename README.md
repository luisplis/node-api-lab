# Laboratorio Práctico

## Realizar operaciones CRUD con Node.js y Express en API Rest-Full

En este laboratorio aprenderá a crear una lista de amigos utilizando Express server mediante una API REST que permita realizar operaciones CRUD sobre datos transitorios. 

Su aplicación deberá manejar contactos con los siguientes detalles: Nombre, Apellidos, Correo electrónico y Fecha de nacimiento -- ¿MongoDB?.

Creará una aplicación con puntos finales API para realizar operaciones API de CRUD como creación, recuperación, actualización y eliminación de contactos con un servidor Express. 

Aprenderá a proporcionar acceso autenticado a los puntos finales. Utilizará cURL y Postman para probar los endpoints implementados.

**Objetivos**:

- Crear puntos finales de API para realizar operaciones de Crear, Recuperar, Actualizar y Eliminar sobre datos transitorios con un servidor Express.

- Implementar la autenticación a nivel de sesión utilizando Tokens Web JSON (JWT) para el acceso autorizado.
